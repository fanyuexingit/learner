# javaWeb

## 1. 概念
* 利用java开发web

### 1.1 作用
实现浏览器和服务器的交互
* 接受请求
* 逻辑处理
* 返回响应

### 1.2 javaWeb 目录结构
* bbs目录代表一个web应用
* bbs目录下的html,jsp文件可以直接被浏览器访问
* WEB-INF目录下的资源是不能直接被浏览器访问的
* web.xml文件是web程序的主要配置文件
* 所有的classes文件都放在classes目录下
* jar文件放在lib目录下

当然也可以使用maven代替lib
